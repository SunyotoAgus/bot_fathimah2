Nuansa Islami.

Dengan API dan tersedianya OpenSource Code, diharapkan memiliki banyak turunan produk yang bermanfaat seluas-luasnya buat ummat Islam khususnya, dan seluruh ummat manusia pada umumnya.

Open Source

Tersedia source code yang bersifat terbuka untuk diunduh, dipelajari dan dikembangkan. Yakni engine bot Fathimah. Silakan di adopsi, disesuaikan dan dimanfaatkan seluas-luasnya untuk kebutuhan masing-masing ataupun bersama-sama kita kembangkan.

Muslim API

Menyediakan berbagai API untuk kebutuhan para developer, utamanya bagi muslim. API yang dapat dimanfaatkan untuk pembuatan aplikasi (desktop, mobile, dll), bot, web based, dan segala macamnya. InsyaAllah.
